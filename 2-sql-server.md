### What is SQL Server?
**SQL Server** is a **Relational Database Management System (RDBMS)** by Microsoft used for storing, retrieving, and managing data in a structured way.

### SQL Server Components
  | Name | Usage |
  | ----------- | ----------- |
  | **SQL Server Database Engine** | The main part - stores, processes, and secures data |
  | **SQL Server Agent** | Schedules jobs, automation tasks |
  | **SQL Server Management Studio (SSMS)** | A graphical tool to connect and manage the server |
  | **SQL Server Reporting Services (SSRS)** | Used for creating reports |

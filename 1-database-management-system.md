### What is DBMS?

**DBMS (Database Management System)** is software used to store data in a structured way and perform operations like:

  - Creating a database
  - Inserting data
  - Updating data
  - Deleting data
  - Retrieving data with queries
  - Managing users and security
  - Ensuring consistency and reliability
